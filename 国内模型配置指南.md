# 🇨🇳 国内模型配置指南

## 推荐模型（按优先级）

### 1️⃣ DeepSeek-V3（推荐首选）
- **性价比最高**：¥0.5/1M tokens（输入），¥2/1M tokens（输出）
- **代码能力强**：专门优化了代码理解和生成
- **API 文档**：https://platform.deepseek.com

**获取 API Key：**
1. 注册 https://platform.deepseek.com
2. 控制台 → API Keys → 创建
3. 新用户送 ¥10 额度

**配置：**
```yaml
REVIEW_MODEL: 'deepseek-chat'
```
GitHub Secrets 添加：`DEEPSEEK_API_KEY`

---

- **阿里出品**：代码专项优化
- **价格**：¥1-2/1M tokens
- **API 文档**：https://dashscope.console.aliyun.com

**获取 API Key：**
1. 注册阿里云账号
2. 开通 DashScope 服务
3. 控制台 → API Key 管理

**配置：**

## 快速开始（以 DeepSeek 为例）

### 1. 获取 API Key
1. 访问 https://platform.deepseek.com
2. 注册/登录
3. 控制台 → API Keys → 创建新 Key
4. 复制 Key（形如 `sk-xxxxx`）

### 2. 添加到 GitHub Secrets
1. 打开 https://github.com/yuzhangCZ/agent-plugin/settings/secrets/actions
2. New repository secret
3. Name: `DEEPSEEK_API_KEY`
4. Value: 你的 DeepSeek API Key

### 3. 修改 Workflow（可选）
当前配置已默认使用 DeepSeek：
```yaml
REVIEW_MODEL: 'deepseek-chat'
```

如果用其他模型，改这一行即可。

### 4. 测试
创建 PR，AI 会自动审查！

## 常见问题

**Q: 国内模型 API 稳定吗？**
A: DeepSeek、通义千问都很稳定，生产环境可用。

**Q: 需要备案吗？**
A: GitHub Actions 调用 API 不需要备案。

**Q: 可以多个模型一起用吗？**
A: 可以，但每次 PR 只能用一种。可以配置不同分支用不同模型。

**Q: 免费额度够用吗？**
A: DeepSeek 新用户送 ¥10，够审查 50-100 个中小 PR。

## 模型切换

想换模型？改两处：

1. `.github/workflows/ai-pr-review.yml`:
```yaml

```

2. GitHub Secrets 添加对应的 API Key
